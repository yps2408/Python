Пример:

- [2, 3, 5, 9, 3] -> на нечётных позициях элементы 3 и 9, ответ: 12

Пример:
- [2, 3, 4, 5, 6] => [12, 15, 16];

Пример:

- [1.1, 1.2, 3.1, 5, 10.01] => 0.20
4.Напишите программу, которая будет преобразовывать десятичное число в двоичное.

Пример:
- 45 -> 101101
- 3 -> 11
- 2 -> 10
